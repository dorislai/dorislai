### 🪄 About Me

我是一位 **產品經理 (Product Manager)**，喜歡待在「使用者需求」與「工程實作」之間的灰色地帶，把模糊的問題整理成清楚的方向。

- 🔭 目前專注於：把使用者洞察轉化為可被衡量的產品決策
- 🧩 擅長：跨團隊溝通、需求拆解、產品路線圖規劃、數據驅動的迭代
- 🌱 持續學習中：SQL 分析、AI 產品設計、使用者研究方法
- 💬 信念：*Good products are obvious in hindsight.*（好的產品事後看都理所當然）
- ☕ 工作之外：一杯咖啡、一隻貓 🐱、還有看不完的待讀清單
